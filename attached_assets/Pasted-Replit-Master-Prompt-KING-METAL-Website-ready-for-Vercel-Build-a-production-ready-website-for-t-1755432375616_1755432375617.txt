Replit Master Prompt — KING METAL (Website ready for Vercel)

Build a production-ready website for the company “KING METAL” (Arabic-first, Egypt-based), using Next.js 14 (App Router) + TypeScript + Tailwind CSS + Framer Motion + shadcn/ui + Lucide icons + next/font/google. The site must be dark, luxurious, fast, and deployable to Vercel without any issues. Integrate Vercel Analytics out of the box.

Key goals

Brand: KING METAL — Slogan: "ملوك الالوميتال فى مصر" (Arabic).

Theme: deep elegant black with subtle metallic/brushed-steel textures and glossy edge highlights (light streaks on edges when hovering/scrolling).

Performance: very fast, responsive (mobile-first), accessible (WCAG), and SEO-ready.

Visual concept: homepage sections look like armored doors (interactive “door panels”) that open/close to reveal content. Think tasteful, realistic metal—not cartoonish.

Arabic RTL by default. Clean English fallbacks allowed for tech text, but main UI is Arabic.

Tech requirements

Next.js 14 (App Router) + TypeScript. Use server components where possible.

Tailwind CSS configured; add custom utilities for metallic gradients and edge highlights (e.g., conic/radial-overlay).

Fonts: use next/font/google → Arabic-friendly (e.g., Tajawal or Cairo) for headings/body; Inter for Latin.

framer-motion for snappy, elegant animations (200–300ms), respecting prefers-reduced-motion.

shadcn/ui for base components (Buttons, Accordion, Card, Dialog, Sheet).

Lucide-react icons (shield, crown, kitchen, wrench, phone, whatsapp, mail).

Vercel Analytics: install @vercel/analytics, import { Analytics } in app/layout.tsx and render <Analytics />.

Images: use Next <Image /> with placeholders and lazy loading. Provide a few abstract, realistic metal textures via CSS (no external URLs required).

SEO: set metadata, Open Graph, robots.txt, sitemap.xml (can use a simple static sitemap). Add JSON-LD LocalBusiness for a metal works company in Egypt.

Deployment: ensure npm run build works with zero config on Vercel. No Node APIs that require custom servers. Include a minimal vercel.json only if necessary.

Styling details

Palette: near-black background (#0b0b0d), gunmetal grays, subtle warm highlights (golden accent for crown/edges). Avoid flat pure black; use layered gradients.

Glossy edge highlight: create a reusable utility that renders a faint animated highlight along borders on hover/in-view (e.g., before: pseudo-element with conic-gradient and low opacity).

Metal texture: CSS-based using layered linear-gradient & noise texture via CSS (no external images required).

Buttons: premium, slightly beveled with thin golden border and subtle inner glow on hover.

Animations: door open/close uses perspective 1200–1600px, rotateY approx 0→-12deg on hover/tap, and a full open state on click/expand (Accordion-like). Respect reduced motion.
